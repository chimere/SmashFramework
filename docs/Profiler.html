Simple, static hierarchical block profiler.

Currently it is hardwired to start measuring when you press P, and dump results to the log when you let go of P. Eventually something more intelligent will be appropriate.

Use it by calling Profiler.enter("CodeSectionName"); before the code you wish to measure, and Profiler.exit("CodeSectionName"); afterwards. Note that Enter/Exit calls must be matched - and if there are any branches, like an early return; statement, you will need to add a call to Profiler.exit() before the return.

Min/Max/Average times are reported in milliseconds, while total and non-sub times (times including children and excluding children respectively) are reported in percentages of total observed time.

final class ProfileInfo
{
    public var name:String;
    public var children:Object={};
    public var parent:ProfileInfo;
    
    public var startTime:int, totalTime:int, activations:int;
    public var maxTime:int=int.MIN_VALUE;
    public var minTime:int=int.MAX_VALUE;
    
    final public function ProfileInfo(n:String, p:ProfileInfo=null)
    {
        name = n;
        parent = p;
    }
    
    final public function wipe():void
    {
        startTime = 0;
        totalTime = 0;
        activations = 0;
        maxTime = int.MIN_VALUE;
        minTime = int.MAX_VALUE;
    }
}
